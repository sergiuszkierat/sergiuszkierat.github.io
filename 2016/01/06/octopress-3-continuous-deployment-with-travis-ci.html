Octopress 3: Continuous Deployment With Travis Ci

Jan 6, 2016

Inspired by Write in Octopress 3.0 and publish posts via travis-ci

You could use GITHUB_TOKEN in a safe way. Travis gives you this possibility through encryption keys feature.

Prerequisites

  • Octopress 3
  • Travis CI

Encrypted credentials

We will generate a GitHub’s personal access token and we will encrypt it.

  • Generate a token by:
    • Go to Tokens
    • Click Generate new token button
    • Write a descriptive Token description
    • Select only public_repo scope
    • Click Generate token
    • Copy the token to your clipboard
  • Install the Travis client
  • Encrypt generated token
  • Put encrypted token into your .travis.yml and verify it

Deploying site

To deploy my Octopress blog, I use a deploy subcommand and generate _deploy.yml on the fly.

Summary

This post is published securely via Travis CI.
